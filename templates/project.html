Mothers at Work

Is it possible to accurately predict if a woman will return to work after having a baby?

✶

Project Intent

The purpose of this project was to see what affects the likelyhood of a woman returning to work after having a baby , with data obtained from OECD.org as well as a survery sent out to the mothers of Perth. Using predicitive analytics to create a model based on survey results, a website was built to provide employers a place to go to assess the likelyhood of their employee returing to work.

✶

Methodology

Firstly, collected data was cleaned and investigated. Maternal empoloyment rates from countries around the world are presented on the Visualisations page.

Secondly, a small scale study was undertaken, asking women on different social media platforms questions about thier current employment status. The results of this were cleaned and used to produce a logistical regression model for predicting if a woman will return to work.

Finally, a website was built, giving the model value to nervous employers!
